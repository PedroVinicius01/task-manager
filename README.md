# Task Manager – Full Stack

Projeto full-stack para gerenciamento de tarefas, com **back-end em Node.js** e **front-end em React**, utilizando **MySQL** como banco de dados.

## Tecnologias
- Node.js
- Express
- MySQL
- JWT
- React
- Fetch API

## Funcionalidades
- Cadastro e login de usuários
- Autenticação com JWT
- CRUD de tarefas
- Cada usuário acessa apenas suas próprias tarefas

##  Como executar

### Back-end
```bash
cd backend/task-manager-api
npm install
npm run dev

Front-end
cd frontend/task-manager-front
npm install
npm start


Acesse:

Front: http://localhost:3000

API: http://localhost:3333
